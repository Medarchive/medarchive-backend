
# Project Name

MedArchive Africa

## Product Type

Decentralized Electronic Health Record (EHR)

## Project Goal

Build a production-ready MVP that enables patients to securely own and manage encrypted health records using the Stellar blockchain.
The MVP demonstrates:

* Patient-owned health identity (Care ID)
* Encrypted health record storage
* Consent-based provider access
* Zero-Knowledge (ZK) proof verification
* Complete audit trail on Stellar Testnet

Success Criteria

* A reviewer should be able to:
* Create account
* Connect Stellar wallet
* Generate Care ID
* Upload encrypted health record
* Store encrypted metadata
* Generate ZK proof
* Request provider access
* Approve access
* Verify proof
* View blockchain audit history

Deliverables
By the end of the sprint, the team should provide:

* Create a Stellar wallet
* Create a Care ID
* Upload an encrypted health record
* Verify record exists on-chain
* Generate a ZK proof
* Request provider access
* Approve consent
* View audit history
* Verify all actions through Stellar testnet transaction hashes

Definition of Done
The MVP is considered complete when a new user can:

1. Register and connect a Stellar wallet.
2. Receive a unique Care ID.
3. Upload an encrypted medical record.
4. Have the encrypted record referenced on Stellar Testnet.
5. Generate and verify a supported ZK proof.
6. Receive and approve a provider access request.
7. Allow a provider to view only the authorized medical fields.
8. Review a complete, immutable audit trail of all actions.
